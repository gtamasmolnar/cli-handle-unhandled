# @gtamo/cli-handle-unhandled

> Custom error handleError for the script crash on unhandled rejections.

## Install

```sh
npm install @gtamo/cli-handle-unhandled
```

## Usage

```ts
import unhandledError from '@gtamo/cli-handle-unhandled';

unhandledError();
```
